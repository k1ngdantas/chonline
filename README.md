# ChOnline - Sistema de Chamados

Dashboard moderno para gestão de chamados (helpdesk) com foco em **usuários**.

## Como rodar

1. Certifique-se de ter **Node.js** instalado.
2. No diretório do projeto, instale as dependências (se necessário) e execute:

```bash
npm install
npm start
```

O servidor HTTP simples irá servir os arquivos estáticos em `public` na porta `3000` (`http://localhost:3000`).
